# NumbatDiffusionSF
!syntax description /Kernels/NumbatDiffusionSF

Implements the weak form of the dimensionless diffusion term
\begin{equation}
\gamma \nabla^2 c,
\end{equation}
where $\gamma$ is the permeability anisotropy.
