# NumbatDiffusion
!syntax description /Kernels/NumbatDiffusion

Implements the weak form of the diffusion term
\begin{equation}
\phi D \nabla^2 c,
\end{equation}
where $\phi$ is the porosity, and $D$ is the diffusion coefficient.
