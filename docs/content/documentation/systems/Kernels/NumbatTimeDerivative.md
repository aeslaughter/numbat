# NumbatTimeDerivative

!syntax description /Kernels/NumbatTimeDerivative

Implements the weak form of the time derivative of concentration
\begin{equation}
\phi \frac{\partial c}{\partial t},
\end{equation}
where $\phi$ is porosity.
