NumNonlinearIterations

Outputs the number of nonlinear iterations

Description

NumNonlinearIterations reports the number of nonlinear iterations in the just-completed solve.

Example Input Syntax


[Postprocessors]
  [./nonlin_it]
    type = NumNonlinearIterations
  [../]
  [./cumulative_nonlin_it]
    type = CumulativeValuePostprocessor
    postprocessor = nonlin_it
  [../]
[]
(../moose/test/tests/postprocessors/cumulative_value_postprocessor/cumulative_value_postprocessor.i)
