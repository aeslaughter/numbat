LagrangeVecFunctionDirichletBC

Imposes the essential boundary condition , where components are calculated with functions.

Description

LagrangeVecFunctionDirichletBC is a generalization of LagrangeVecDirichletBC which imposes possibly temporally- and spatially-dependent values for the Lagrange vector components through MOOSE Function objects on degrees of freedom that fall on the boundary defined by the boundary parameter.
