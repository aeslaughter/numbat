List of available objects

Numbat provides a number of MOOSE objects to simplify the process of simulating density-driven convective mixing in porous media.

AuxKernels

In Moose, an AuxKernel is used to enable calculation of a quantity that can be explicitly computed from the solution variables or material properties. An AuxKernel acts on an AuxVariable, and hence returns a single value per node/element.

    Numbat Objects
  • keyboard_arrow_downNumbatDarcyVelocity: Calculates Darcy velocity
    Calculates Darcy velocity
  • keyboard_arrow_downNumbatDarcyVelocitySF: Calculates Darcy velocity
    Calculates Darcy velocity

BCs

Boundary conditions are a fundamental MOOSE system, with several common boundary conditions already available in the MOOSE framework. Numbat provides two additional boundary conditions that are useful in density-driven convection mixing. In particular, the NumbatPerturbationBC can be used to seed the initial instability by applying a random noise to the concentration value at the boundary.

    Numbat Objects
  • keyboard_arrow_downNumbatDiffusionDGBC: Dirichlet boundary condition for use with NumbatDiffusionDG
    Dirichlet boundary condition for use with NumbatDiffusionDG
  • keyboard_arrow_downNumbatPerturbationBC: Dirichlet boundary condition that includes a random perturbation
    Dirichlet boundary condition that includes a random perturbation

DGKernels

DGKernels are used to implement the governing equations using a Discontinuous Galerkin formulation.

    Numbat Objects
  • keyboard_arrow_downNumbatConvectionDG: Discontinuous Galerkin upwinded convection
    Discontinuous Galerkin upwinded convection
  • keyboard_arrow_downNumbatDiffusionDG: Diffusion kernel for the Discontinuous Galerkin method
    Diffusion kernel for the Discontinuous Galerkin method

ICs

Several initial conditions are provided in the MOOSE framework. Numbat provides an additional initial condition that applies a noise to the initial concentration profile to seed the instability.

    Numbat Objects
  • keyboard_arrow_downNumbatPerturbationIC: Perturbed initial condition that can be used to initiate convection
    Perturbed initial condition that can be used to initiate convection

Kernels

The governing equations are encapsulated in the Numbat Kernels.

    Numbat Objects
  • keyboard_arrow_downNumbatConvection: Convection of concentration with velocity given by Darcy’s law
    Convection of concentration with velocity given by Darcy’s law
  • keyboard_arrow_downNumbatConvectionSF: Convection of concentration with velocity given by Darcy’s law using the streamfunction formulation
    Convection of concentration with velocity given by Darcy’s law using the streamfunction formulation
  • keyboard_arrow_downNumbatDarcy: Darcy’s law
    Darcy’s law
  • keyboard_arrow_downNumbatDarcySF: Darcy’s law for the streamfunction formulation
    Darcy’s law for the streamfunction formulation
  • keyboard_arrow_downNumbatDiffusion: Diffusion kernel with porosity
    Diffusion kernel with porosity
  • keyboard_arrow_downNumbatDiffusionSF: Diffusion kernel for the streamfunction formulation
    Diffusion kernel for the streamfunction formulation
  • keyboard_arrow_downNumbatTimeDerivative: Time derivative kernel
    Time derivative kernel

Materials

The Materials system provides fluid and rock properties to the Kernels.

    Numbat Objects
  • keyboard_arrow_downNumbatDensity: This Material calculates the density of the fluid with given concentration
    This Material calculates the density of the fluid with given concentration
  • keyboard_arrow_downNumbatDiffusivity: This Material provides a diffusivity material property
    This Material provides a diffusivity material property
  • keyboard_arrow_downNumbatPermeability: This Material provides a constant permeability material property
    This Material provides a constant permeability material property
  • keyboard_arrow_downNumbatPermeabilityFromVar: This Material provides the permeability tensor given by the input variables
    This Material provides the permeability tensor given by the input variables
  • keyboard_arrow_downNumbatPorosity: This Material provides a porosity material property
    This Material provides a porosity material property
  • keyboard_arrow_downNumbatViscosity: This Material provides a viscosity material property
    This Material provides a viscosity material property

MeshModifiers

Numbat provides a MeshModifier to enable a regular, orthogonal mesh to be biased at a particular boundary, with a specified resolution.

    Numbat Objects
  • keyboard_arrow_downNumbatBiasedMesh: MeshModifier to bias the mesh with a specified initial resolution
    MeshModifier to bias the mesh with a specified initial resolution

Postprocessors

A Postprocessor performs an explicit calculation and returns a single value. In Numbat, Postprocessors to calculated the flux over a boundary or total mass of solute in the model are provided, as these quantities are of significant interest in practice.

    Numbat Objects
  • keyboard_arrow_downNumbatEffectivePermeability: Calculates effective permeability of equivalent homogeneous model
    Calculates effective permeability of equivalent homogeneous model
  • keyboard_arrow_downNumbatRayleighNumber: Calculates Rayleigh number
    Calculates Rayleigh number
  • keyboard_arrow_downNumbatSideFlux: Calculates the mass flux over the boundary
    Calculates the mass flux over the boundary
  • keyboard_arrow_downNumbatSideFluxSF: Calculates the dimensionless mass flux over the boundary for the streamfunction formulation
    Calculates the dimensionless mass flux over the boundary for the streamfunction formulation
  • keyboard_arrow_downNumbatTotalMass: Calculates the total mass in model
    Calculates the total mass in model
  • keyboard_arrow_downNumbatTotalMassSF: Calculates the total dimensionless mass in model for the streamfunction formulation
    Calculates the total dimensionless mass in model for the streamfunction formulation
