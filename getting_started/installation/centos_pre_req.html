Prerequisites

Install the following using yum


  sudo -E yum install gcc \
gcc-c++ \
make \
freeglut-devel\
m4 \
blas-devel \
lapack-devel \
gcc-gfortran \
libX11-devel

Download and install one our redistributable packages according to your version of CentOS

  • CentOS 7: moose-environment-1_CentOSLinux-7.4.1708_x86_64.rpm
