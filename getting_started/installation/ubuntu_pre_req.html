Prerequisites

Install the following using apt-get


sudo apt-get install build-essential \
gfortran \
tcl \
git \
m4 \
freeglut3 \
doxygen \
libblas-dev \
liblapack-dev \
libx11-dev \
libnuma-dev \
libcurl4-gnutls-dev \
zlib1g-dev \
libhwloc-dev

Download and install one of our redistributable packages according to your version of Ubuntu.

  • Ubuntu 18.04: moose-environment-2_Ubuntu-18.04_x86_64.deb

  • Ubuntu 16.04: moose-environment-74_Ubuntu-14.04_x86_64.deb
