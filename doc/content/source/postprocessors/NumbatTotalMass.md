# NumbatTotalMass

!syntax description /Postprocessors/NumbatTotalMass

The total mass of solute in the system is given by
\begin{equation}
M = c \phi \rho,
\end{equation}
where $c$ is the concentration, $\phi$ is porosity, and $\rho$ is the fluid density.
